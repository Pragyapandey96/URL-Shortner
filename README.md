# 🔗 URL Shortener

A simple URL Shortener built using Node.js that converts long URLs into short and manageable links.

## 📌 About the Project
This project allows users to paste a long URL and generate a short URL which redirects to the original link. It is useful for sharing long links in a clean and simple way.

## 🚀 Features
- Convert long URLs into short URLs
- Redirect short URL to the original link
- Unique short ID generation
- Stores URL data in database
- Simple and fast API

## 🛠️ Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- NanoID / ShortID (for generating short links)
